# Estilos.css — Estilização da Aplicação

## Função do arquivo

O `estilos.css` define toda a aparência visual do projeto.

Ele controla:

* cores
* tamanhos
* espaçamentos
* alinhamentos
* responsividade
* aparência dos componentes

---

# Estilo Geral
* fundo escuro
* tipografia principal
* cor global do texto

### Objetivo:

Criar uma interface acolhedora e confortável visualmente.

---

# Container Principal

Responsável por:

* centralizar o conteúdo
* limitar largura máxima
* adicionar espaçamento interno

### Benefícios:

* melhor leitura
* layout organizado
* adaptação a telas maiores

---

# Input Box

Card principal de escrita.

### Características:

* fundo escuro
* bordas arredondadas
* espaçamento interno

---

# Textarea

Campo de entrada de texto.

### Define:

* largura total
* altura fixa
* padding interno
* remoção da borda padrão

### Objetivo:

Melhorar conforto durante escrita.

---

# Linha de Controles

Usa Flexbox para alinhar:

* select
* botão publicar

### Benefícios:

* alinhamento horizontal
* espaçamento automático

---

# Botões

Estilo global dos botões.

### Características:

* fundo verde
* cantos arredondados
* cursor pointer
* texto branco

---

# Feed

Área das publicações.

### Utiliza:

* Flexbox em coluna
* espaçamento entre cards

---

# Cards

Representa cada desabafo.

### Possui:

* fundo escuro
* padding
* bordas arredondadas

---

# Sentimentos

```css
.triste
.raiva
.alivio
```

Classes responsáveis pelas cores laterais dos cards.

### Objetivo:

Representar visualmente o sentimento do usuário.

---

# Reações

Container dos botões de apoio emocional.

### Organiza:

* alinhamento horizontal
* espaçamento entre botões

---

# Responsividade

O CSS utiliza:

* largura máxima
* espaçamentos flexíveis
* Flexbox

### Resultado:

Melhor adaptação em diferentes tamanhos de tela.
