# Index.html — Estrutura da Página

# Estrutura Geral

O corpo da página contém:

* Cabeçalho
* Área de desabafo
* Controles
* Feed de publicações

---

### Elementos:

* Título do site
* Aviso informativo

### Objetivo:

Mostrar ao usuário o propósito da plataforma e reforçar que o sistema não substitui ajuda profissional.

---

# Input Box
Área principal de interação do usuário.

### Contém:

* Título da seção
* Campo de texto (`textarea`)
* Seleção de sentimento (`select`)
* Botão de publicação
* Feedback visual

---

## Textarea

Campo onde o usuário escreve o desabafo.

### Características:

* Entrada multilinha
* Placeholder acolhedor
* Identificação única (`id="texto"`)

---

## Selecionar sentimento

Permite selecionar o estado emocional.

### Opções:

* Tristeza
* Raiva
* Alívio

---

## Botão Publicar

Aciona o envio do desabafo.

### Função:

* Capturar dados
* Criar publicação
* Atualizar feed

---

# Feedback

Área usada para mensagens dinâmicas.

### Exemplos:

* “Desabafo publicado”
* “Você não está sozinho”

---

# Ações

Área de controle da aplicação.

### Contém:

* Botão “Apagar tudo”

---

# Top Bar

Barra superior do feed.

### Contém:

* Filtro de sentimentos
* Contador de desabafos

---

# Feed

Área onde os cards são renderizados dinamicamente pelo JavaScript.

### Características:

* Atualização automática
* Ordem do mais recente para o mais antigo
* Conteúdo gerado dinamicamente

---
